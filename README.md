# mol

### setup

create a virtual env. install numpy, scipy

### run tests using pytest
```sh
pytest -v

```
or run the `pytest_script.sh`.

### Example run

- Edit the `input.json` with the parameters you want
- xyz files are in the xyz/ directory

Run,
```sh
python run.py
```
